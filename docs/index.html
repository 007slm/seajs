FlyScript is a CommonJS module loader for the web. It's designed to load modules and get out of the way. Here's how it works: 

<script src="flyscript.js"></script>

<script>

FlyScript.load("./main", function(main)
{
    main.doSomething();
});

</script>

Don't know what a CommonJS module is? FlyScript loads wrapped modules. They look something like this: 

module.declare(function(require) {

    // Load dependencies like this:
    var whateva = require("./whateva");

    // Export symbols like this:
    return {
        flyscript: "rocks"
    };

});
