全局配置

可以用 module.config 对 SeaJS 进行全局配置： 

module.config({
  base: 'path/to/seajs/1.0.0/',
  main: './init',
  prefix: {
    'biz': 'path/to/biz/'
  }
});
  • base - module.js 所在目录。默认会自动解析加载路径来获取。
  • main - 入口模块的 id .
  • prefix - 全局 prefix 设置。

prefix

当模块标识很长时，可以通过 prefix 设置来简化 require 调用处的代码：

module.config({
  prefix: {
    'biz': 'path/to/biz'
  }
});

module.declare(function(require, exports, module) {
    var a = require('{biz}/a');
    // 等价于：var a = require('path/to/biz/a');
});
