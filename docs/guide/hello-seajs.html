Hello, SeaJS!

向未知的宇宙文明，用地球语言说出一句“你好”，是人类的一个梦。在这里，我们来看看如何给 SeaJS 打招呼： 

/*** File: index.html ***/
<!doctype html>
<html>
<head>
<meta charset="utf-8"/>
<title>Hello, SeaJS!</title>
</head>
<body>
<script src="sea.js"></script>
<script>
  seajs.use('./hello', function(hello) {
    hello.sayHello();
  });
</script>
</body>
</html>

上面的页面代码中，通过 script 标签引入了 sea.js 文件，然后通过 seajs.use 就可以加载任意模块了。hello 模块的文件内容为： 

/*** File: hello.js ***/
define(function(require, exports) {

  exports.sayHello = function() {
    alert('你好，SeaJS!');
  };

});

将 index.html, sea.js 和 hello.js 放在同一目录，我们就学会给 SeaJS 打招呼了： 在线演示 。

还有一个稍微复杂点的例子： Spinning Icons. 请查看源码一窥究竟。

注意：必须通过 Web 服务器访问才能正确加载，不支持本地文件直接打开。
