
Install on Mac OS X
--------------------

 - brew install node
 - sudo vi /etc/paths, and add /path/to/seajs/tools/bin


Install on Windows
-------------------

 - add X:\\path\\to\\seajs\\tools\\bin to your %PATH%


Usage
------

    sbuild --help
    sbuild [--combo] a.js
    sbuild [--combo] a.js b.js
    sbuild [--combo] *.js
    sbuild [--combo] [-r] some_directory
    sbuild clear
