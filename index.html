A new kind of JavaScript module loader

Here bellow is an example of a JavaScript module which provides a sayHello method:

define(function(require, exports, module) {
  exports.sayHello = function() {
    alert('Hello from module: ' + module.id);
  };
});

To use the above module, put the code into a file example.js and launch it with a html file: 

<script src="sea.js"></script>
<script>
  seajs.use('./example', function(example) {
    example.sayHello();
  });
</script>

Please refer to the API documentation for more details.

About

The sea, holding water from thousands of rivers, is huge owing to its capacity.

SeaJS is a new kind of JavaScript module loader that makes it easy to build scalable web applications. It is designed to change the way that you organize JavaScript.
