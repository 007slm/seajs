Summary


1. Chrome、Safari、Firefox、Opera 下，script 添加到 fragment 中时不会执行，添加
   到 DOM 树中时才会下载并执行，并且执行上下文是全局环境，一切符合预期。

2. IE6-9 下，在 script 添加到 fragment 中时，就会触发下载并执行。执行时上下文
   是 fragment，相当于 with(fragment) { (function() { script code })() }

3. 标准浏览器包括 IE9+ 下，fragment 没有 getElementById 等方法，但在 IE6-8 下有。

4. IE 下，利用 script 在 fragment 中的特性，可以干不少事情，嘿嘿。
